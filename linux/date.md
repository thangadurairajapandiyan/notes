# date command

### date command without any parameters
```
#date
12 Jan 2021 16:19:37
```

### date with specific format
```
#date +'%d-%m-%y'
12-01-21

#date +'%d-%m-%Y'
12-01-2021
```
here,
%D - date ->(05/18/2021)
%d - day - >18
%y- year in short form ->21
%Y - year in full form -> 2021
%m- month -> 05
%M - minute -> minute in current time
%H - hour
%S - Sec
